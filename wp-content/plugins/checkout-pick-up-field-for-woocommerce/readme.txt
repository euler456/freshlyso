Checkout Pick Up Field for WooCommerce - The WooCommerce checkout manager plugin to customize pick up checkout fields on your WooCommerce checkout page.

== Description ==

= Checkout Pick Up Field for WooCommerce =
Checkout Pick Up Field Editor for WooCommerce plugin provides an easy way to customise your pick up fields displayed on WooCommerce checkout page. This plugin requires WooCommerce to be installed, activated, and configured.

= Plugin Features =
* **Add pick up fields.** Adding pick up fields to the WooCommerce checkout page.
* **Option fields.** You can enable or disable option fields.
* Available field types are
 * Date
 * Time slot
* **Display in Order Details page.** Pick up fields are available in in Order Details page.

== Installation ==
= Minimum Requirements =
* WooCommerce 3.0 or greater
* WordPress 4.9 or greater

= Automatic installation =
1. Log in to your WordPress dashboard.
2. Navigate to the Plugins menu, and click "Add New".
3. Search and locate 'Checkout Pick Up Field for WooCommerce' plugin.
4. Click 'Install Now', and WordPress will take it from there.

= Manual installation =
Manual installation method requires downloading the 'Checkout Pick Up Field for WooCommerce' plugin and uploading it to your web server via your FTP application. The WordPress codex contains [instructions on how to do this here](https://wordpress.org/support/article/managing-plugins/#manual-plugin-installation).
